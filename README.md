# Proyecto: Aplicación de Consola - Módulo 3

## Fundamentos de programación en Javascript

### 📝 Descripción

- [cite_start]El objetivo es crear una herramienta de consola que permita a los usuarios realizar operaciones matemáticas, gestionar estructuras de datos y aplicar lógica de control[cite: 12, 14, 15].

### 🛠️ Requerimientos Técnicos

- [cite_start]**Ejecución:** Debe ejecutarse en la consola del navegador[cite: 22].
- [cite_start]**Entrada de datos:** Uso de `prompt()` o variables predefinidas[cite: 23, 24].
- [cite_start]**Operaciones:** Al menos tres operaciones diferentes usando funciones[cite: 25, 26].
- [cite_start]**Control:** Implementación de estructuras condicionales (if, switch) y de repetición (for, while)[cite: 27, 33].
- [cite_start]**Estructuras:** Uso de arreglos y objetos para almacenar y manipular datos[cite: 30, 34].
- [cite_start]**Calidad:** Código modularizado, con validaciones de entrada y comentarios explicativos[cite: 32, 35, 93].

### 🚀 Funcionalidades Principales

1. [cite_start]**Sintaxis básica:** Uso de `console.log()`, `alert()` y `prompt()`[cite: 46, 47, 48].
2. [cite_start]**Lógica Matemática:** Definición de funciones para suma, resta, multiplicación y división[cite: 58, 74].
3. [cite_start]**Control de Flujo:** Uso de sentencias condicionales para evaluar situaciones[cite: 60].
4. [cite_start]**Manipulación de Datos:** Recorrido de arreglos y uso de objetos con métodos propios[cite: 67, 82, 83].
