# Proyecto: Aplicación de Consola - Módulo 3

## Fundamentos de programación en Javascript

### 📝 Descripción

El objetivo es crear una herramienta de consola que permita a los usuarios realizar operaciones matemáticas, gestionar estructuras de datos y aplicar lógica de control.

### 🛠️ Requerimientos Técnicos

- **Ejecución:** Debe ejecutarse en la consola del navegador.
- **Entrada de datos:** Uso de `prompt()` o variables predefinidas.
- **Operaciones:** Al menos tres operaciones diferentes usando funciones.
- **Control:** Implementación de estructuras condicionales (if, switch) y de repetición (for, while).
- **Estructuras:** Uso de arreglos y objetos para almacenar y manipular datos.
- **Calidad:** Código modularizado, con validaciones de entrada y comentarios explicativos.

### 🚀 Funcionalidades Principales

1. **Sintaxis básica:** Uso de `console.log()`, `alert()` y `prompt()`.
2. **Lógica Matemática:** Definición de funciones para suma, resta, multiplicación y división.
3. **Control de Flujo:** Uso de sentencias condicionales para evaluar situaciones.
4. **Manipulación de Datos:** Recorrido de arreglos y uso de objetos con métodos propios.

### 📂 Estructura del Repositorio

- `index.html` / `script.js`: Archivos principales de la aplicación.
- `README.md`: Documentación del proyecto.

### 🎓 Sobre el Proyecto

Este trabajo fue desarrollado siguiendo las buenas prácticas de programación, priorizando la legibilidad, la organización del código y el uso de funciones reutilizables.
